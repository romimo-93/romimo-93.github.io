Leaflet Challenge - Add image of script

Leaflet and mapbox vizualization of earthquakes and faultlines around the world

  • Category: Leaflet in Javascript
  • Client: Class
  • Project date: March 30, 2021
  • Project URL: http://www.romi-sinnigen.com/leaflet-challenge/

The elements of this code allow for interactivity within the map. The map displays 2 main datasets: one for the past 7 days of earthquake activity, and the other for major faultlines. The goal is to create a visual connection between earthquake activity and faultline location. Each earthquake location has a popup for exact location and magnitude. The map can be set to 3 different layers, satelite, light, and dark.
